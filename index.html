Symmetric Encryption Demo (AES-GCM)

Learn how password-based symmetric encryption works in your browser using the Web Crypto API.

How this works

  1. Generate a random 16-byte salt and 12-byte IV.
  2. Derive a 256-bit AES-GCM key from your password using PBKDF2 (SHA-256).
  3. Encrypt plaintext with the key and IV to produce ciphertext.
  4. To decrypt, provide the same password, salt, and IV.

All crypto happens locally in your browser via the Web Crypto API.
